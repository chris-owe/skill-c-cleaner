快速上手

点击 📋 一键复制命令

🔍 完整分析

.\analyze.ps1
控制台输出，实时查看 C 盘健康状况

📝 生成 Markdown 报告

.\analyze.ps1 -OutputFormat markdown
生成 AI 决策报告，保存为 .md 文件

🎯 指定类别扫描

.\analyze.ps1 -Categories "C,D"
只扫描 开发缓存(C) + 浏览器(D)

📊 JSON 输出

.\analyze.ps1 -OutputFormat json
输出 JSON 格式，可被其他工具消费

🧹 安全清理（预览模式）

.\cleaners\clean-safe.ps1 -WhatIf
仅预览不删除，确认无误后再执行

🗑️ 安全清理（执行）

.\cleaners\clean-safe.ps1 -ReallyDelete
