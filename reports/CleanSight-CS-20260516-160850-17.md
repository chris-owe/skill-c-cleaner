# 四、如何使用本报告

## 第一步：从第一优先级开始
第一优先级全部是零风险的可安全清理项，放心执行：
```powershell
# 预览将要删除的内容（不会真正删除）
.\cleaners\clean-safe.ps1 -WhatIf

# 确认后正式执行
.\cleaners\clean-safe.ps1 -ReallyDelete
```

## 第二步：处理第二优先级
第二优先级需要你逐项确认，建议先创建系统还原点：
```powershell
# 创建系统还原点
.\safety\snapshot-before-after.ps1 -CreateRestorePoint
# 逐项确认后清理
.\cleaners\clean-deep.ps1
```

## 第三步：迁移（释放最大空间）
如果开发缓存或 Docker/WSL 占用较大，迁移到 D 盘比删除更好：
```powershell
# 迁移开发工具缓存到 D 盘
.\migrators\migrate-dev-caches.ps1 -Tools npm,pip -TargetDrive D -BackupFirst
# 迁移 WSL/Docker 数据到 D 盘
.\migrators\migrate-wsl-docker.ps1 -TargetDrive D -BackupFirst
```

## 第四步：长期预防
设置环境变量让开发工具缓存直接存到 D 盘，一劳永逸：
```powershell
[System.Environment]::SetEnvironmentVariable('npm_config_cache', 'D:\DevCaches\npm', 'User')
[System.Environment]::SetEnvironmentVariable('PIP_CACHE_DIR', 'D:\DevCaches\pip', 'User')
# 重启终端后生效
```
